Introduction to C

C is a programming language developed at AT & T's Bell Laboratories of USA in 1972. It was designed and written by a man named "DANNIS RITCHIE". C is the most reliable , simple and easy to use languages. This contains the basic of any languages like use and work of Pointers, memory allocation, memory distribution. It is very easy to use and understand C language. Whenever you start to learn any language you should start with C language because you will be clear with the concept of important topics like pointers, structure, union, linked lists, etc. We will learn from basic to advance in this course.

Some important Tips for those, who Is Learning For First time

  • Please don't recite (ratta) the whole code.
  • Keep practicing whether it's a example or any questions
  • For the first time if you don't under stand any thing you will feel demotivated, lazy, it is quite common but fill yourself up with positive thoughts. Nothing is impossible my friend.
  • If one topic is done do the exercise question at the same time
  • Don't be panic, it is a code not a song which you will learn by listen one time.
  • Give time small code can take much time. You cannot forget once your concept is clear.
